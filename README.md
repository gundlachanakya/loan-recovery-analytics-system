Loan Recovery Analytics System
📌 Project Overview

This project focuses on optimizing vehicle loan recovery operations using Data Analytics and Machine Learning techniques. The study was conducted for S.A. Associates, a vehicle loan recovery agency based in Siddipet, Telangana.

The project analyzes real-world loan recovery data to identify patterns affecting recovery success, improve operational efficiency, and support data-driven decision-making.

🎯 Objectives
Perform Exploratory Data Analysis (EDA) on loan recovery data
Analyze agent performance and recovery efficiency
Identify factors influencing recovery success
Segment loans using clustering techniques
Predict loan recovery probability using Machine Learning
Forecast commission earnings for operational planning
Provide business recommendations to improve recovery rates

📌 Key Findings
Overdue Days is the strongest predictor of recovery failure.
Experienced agents achieve significantly higher recovery rates.
High-risk loans have much lower recovery probability.
70% of successful recoveries occur within 30 days.
Machine Learning models achieved:
Random Forest Accuracy: ~80–85%
Linear Regression R² Score: ~0.82–0.88
